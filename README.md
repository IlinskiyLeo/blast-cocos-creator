Прототип игры в жанре "Blast", выполненный как тестовое задание.

## Технологии
- TypeScript
- Cocos Creator 2.4

## Игровая механика

### Основное
- Игровое поле 9×9 с цветными тайлами
- При клике на тайл сгорает вся группа прилегающих тайлов того же цвета
- Минимальная группа для сжигания — 2 тайла
- После сжигания тайлы падают вниз, пустые места заполняются новыми
- Цель — набрать 500 очков за 15 ходов

### Очки
- 10 очков за каждый уничтоженный тайл

### Супер-тайлы
- **Ракета** — появляется при уничтожении группы 5-6 тайлов, сжигает всю строку или столбец
- **Бомба** — появляется при уничтожении группы 7+ тайлов, сжигает область 3×3
- Супер-тайлы поддерживают цепные реакции

### Бустеры
- **Бомба** — активирует взрыв 3×3 в выбранной клетке (5 зарядов)
- **Телепорт** — меняет два тайла местами (5 зарядов)

### Автоперемешивание
- Если нет доступных ходов — поле перемешивается автоматически
- Максимум 3 перемешивания, после — проигрыш
